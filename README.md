# Valorant Gun Skins Showcase

A modern website to showcase Valorant gun skins with React frontend and Node.js backend.

## Features
- Aesthetic gun skin gallery with clear image display
- Weapon-specific pages for different gun categories
- Detailed skin information with special feature indicators
- Color variants and animation indicators
- High-resolution image modal view
- Modern Valorant-inspired design

## Project Structure
```
valorantgunskins/
├── backend/          # Node.js Express server
├── frontend/         # React application
└── README.md
```

## Development
- Backend: Node.js + Express
- Frontend: React + modern tooling
- Responsive design for all devices

## Getting Started
1. Install backend dependencies: `cd backend && npm install`
2. Install frontend dependencies: `cd frontend && npm install`
3. Start backend: `cd backend && npm run dev`
4. Start frontend: `cd frontend && npm start`
